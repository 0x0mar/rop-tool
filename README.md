ropc
====

Tool for finding gadgets in ELF32 binaries.


=== OPTIONS

```
Usage : ropc [options]
Options : 
  -s <string>    Search string in memory.
  -g             Search gadgets.
  -f <file>      Search on file.
  -d <depth>     Depth for gadgets searching.
  -b <bad>       Bad chars on address.
  -0             Genere stage0.
  -F             Filter gadgets.
  -N             No colors.
  -h             Print help.
  -v             Print current version.
```
